# Joseph Maina — Portfolio

Personal portfolio for **Joseph Maina**, Full-Stack Engineer (Eldoret, Kenya).

Built with **React**, **Vite**, and **Tailwind CSS v4**.

**Live repo:** [github.com/HIM-coder001/MY-PORTFOLIO](https://github.com/HIM-coder001/MY-PORTFOLIO)

## Quick start

```bash
npm install
npm run dev
```

Open the URL shown in the terminal (typically `http://localhost:5173`).

## Customize

| File | What to update |
|------|----------------|
| `src/data/siteConfig.js` | About text, email, GitHub, LinkedIn, resume URL |
| `src/assets/hero.jpg` | Hero image |
| `public/resume.pdf` | Your actual resume PDF |
| `src/data/projects.js` | Project copy, metrics; set `liveUrl` after deploy to enable Live Demo |

## Build for production

```bash
npm run build
npm run preview
```

Deploy the `dist/` folder to Vercel, Netlify, GitHub Pages, or any static host.

## Publish to GitHub

From the project folder (requires [Git](https://git-scm.com/download/win)):

```bash
git init
git add .
git commit -m "Initial commit: portfolio site"
git branch -M main
git remote add origin https://github.com/HIM-coder001/MY-PORTFOLIO.git
git push -u origin main
```

If the remote already exists, use `git remote set-url origin https://github.com/HIM-coder001/MY-PORTFOLIO.git` instead of `git remote add`.

Sign in when prompted (GitHub username + [personal access token](https://github.com/settings/tokens) as password).
